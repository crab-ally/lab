# 통합 AI 안전 순찰 로봇 시뮬레이션 시스템 (MuJoCo 3.6.0 기반)

버전: 1.0
플랫폼: Python 3.10+, MuJoCo 3.6.0
작성자: 20년 차 로보틱스 시뮬레이션 전문가

## 1. 프로젝트 개요 (Overview)
본 프로젝트는 산업 현장의 사각지대를 해소하고 중대재해를 예방하기 위해, 자율주행 순찰 로봇이 디지털 트윈 환경에서 실시간으로 위험(PPE 미착용, 장애물, 충돌, 화재)을 감지하고 대응하는 통합 시뮬레이션 시스템 구축을 목표로 합니다.

## 2. 시나리오 통합 기능 요구사항 (Functional Requirements)

### 2.1 자율 순찰 및 시스템 자가 진단 (Scenario 1 기반)

- 자가 진단 (Self-Diagnosis): 순찰 시작 전 mjData 구조체를 통해 LiDAR(Range sensor), 카메라, 배터리 상태를 체크하고 정상 동작 여부를 반환해야 합니다.

- 자율주행 주행: 사전에 구축된 실내 지도(SLAM Map)를 기반으로 Waypoint를 생성하고, mj_step() 엔진을 통해 오차 범위 10mm 이내의 정밀 주행을 수행합니다.

- 동적 경로 재탐색: 주행 중 장애물 감지 시 mjData.contact 데이터를 분석하여 물리적 충돌 전 우회 경로를 실시간으로 생성합니다.

- 순찰 종료 및 복귀 조건: 모든 Waypoint 순찰을 완료하거나, 순찰 중 배터리 잔량이 임계치(예: 15%) 이하로 떨어질 경우 순찰을 즉시 종료하고 충전 도킹 스테이션으로 자동 복귀해야 합니다.

### 2.2 지능형 부딪힘 및 넘어짐 위험 감지 (Scenario 2 기반)

- 다중 객체 추적 (MOT): 지게차와 작업자 등 움직이는 객체에 고유 ID를 부여하고 이동 방향 벡터와 속도를 칼만 필터(Kalman Filter) 알고리즘으로 추적합니다.

- 충돌 위험 예측 (TTC): 객체 간의 mjData.geom_xpos 간격을 계산하여 충돌 임계 시간(TTC)이 설정치 이하일 경우 즉각 비상 정지(E-Stop) 및 경고 로직을 실행합니다.

- 전도(넘어짐) 감지: 작업자의 자세(Posture)를 분석하여 바닥에 누워 있는 상태가 일정 시간 지속되면 사고 발생으로 판단합니다.

### 2.3 비전 AI 기반 안전 수칙 및 화재 감지 (Scenario 3, 4 기반)

- 보호구(PPE) 실시간 검증: 시뮬레이션 내 작업자 모델의 특정 태그(안전모, 안전조끼)를 Python OpenCV와 연동하여 인식하고, 미착용 시 음성 경고 알람을 트리거합니다.

- 초기 화재 및 고온 징후 탐지: 가상 열화상 센서의 픽셀 온도 데이터가 임계치를 초과하거나 불꽃 형상이 감지되면 비상 사이렌을 가동합니다.

### 2.4 데이터 로깅 및 안전 리스크 맵 최적화 (Scenario 5 기반)

- 위험 데이터 자동 로깅: 모든 위험 상황 발생 시 시간, 좌표, 위험 유형, 스냅샷 이미지를 CSV 또는 JSON 형태로 자동 저장합니다.

- 안전 리스크 맵(Heatmap): 누적된 로그 데이터를 분석하여 사고 발생 빈도가 높은 '핫스팟' 구역을 시각화하고 순찰 경로 최적화에 반영합니다 [이전 대화 시나리오 5].

## 3. 기술 사양 (Technical Specifications)

### 3.1 시뮬레이션 엔진 및 제어

- 물리 엔진: MuJoCo 3.6.0 (MJCF 포맷 활용)
- 제어 루프: Python 기반 고속 제어 (최소 100Hz 이상 권장)
- 센서 융합: LiDAR(Point Cloud), RGB-D 카메라, IMU 센서 데이터를 융합하여 환경 인식 정밀도를 높임.

### 3.2 통신 및 인터페이스

- 엣지 컴퓨팅: PPE 인식 및 충돌 회피 등 즉각적 판단은 로봇 로컬 환경(Edge)에서 처리하여 지연 시간을 500ms 이내로 단축합니다.
- 원격 관제 스트리밍: 5G/LTE망 시뮬레이션을 통해 현장 영상과 로봇 상태를 관제 대시보드에 실시간 전송합니다.

## 4. 성공 지표 (KPI)
- 감지 정확도: 시뮬레이션 내 설정된 위험 상황에 대해 95% 이상의 식별 성공률 달성 [PRD 성공 지표].
- 충돌 제로화: 지능형 부딪힘 사고 예방 로직 적용 시 이동 객체 간 물리적 충돌 발생 0건.
- 가동 지속성: 자동 충전 및 도킹 기능을 통해 24시간 상시 순찰 체계 유지.
