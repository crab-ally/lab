# AI 위험예측 안전 순찰 로봇 시뮬레이션 시스템

버전: 1.0 | 플랫폼: Python 3.10+, MuJoCo 3.6.0

## 1. 프로젝트 개요 (Overview)

본 프로젝트는 산업 현장의 사각지대를 해소하고 중대재해를 예방하기 위해, 자율주행 순찰 로봇이 실시간으로 위험(PPE 미착용, 장애물, 충돌, 화재)을 감지하고 대응하는 디지털 트윈 시스템을 구축하는 것을 목표로 합니다.

## 2. 기술 스택 (Technical Stack)

Physics Engine: MuJoCo 3.6.0 (MJCF를 활용한 고정밀 물리 접촉 및 센서 시뮬레이션)
Programming: Python (로봇 제어 알고리즘 및 AI 비전 로직 통합)
Environment: 스마트 팩토리 디지털 트윈 (XML/MJCF 기반)
Key Libraries: mujoco, numpy, opencv-python (영상 분석 시뮬레이션), pandas (데이터 로깅)

## 3. 시나리오 통합 요구사항 (Functional Requirements)

### 3.1 자율 순찰 및 시스템 진단 (Scenario 1 기반)

1. 자가 진단 (Self-Diagnosis): 순찰 시작 전 mjData 구조체를 통해 LiDAR(Range sensor), 카메라, 배터리 상태를 체크하고 '정상' 신호를 반환해야 합니다.

2. 자율주행: 사전 정의된 Waypoint를 따라 mj_step()을 실행하며, SLAM 데이터를 기반으로 오차 범위 10mm 이내의 정밀 주행을 시뮬레이션합니다.

### 3.2 물리 기반 위험 감지 (Scenario 2 & 3 기반)

1. 넘어짐(전도) 위험 감지: 통로 내 적치물(Geom)을 감지하고, 보행 방해 여부를 물리적으로 분석합니다. MuJoCo의 접촉 센서(Contact Sensor)를 활용하여 장애물과의 임계 거리를 실시간으로 도출합니다.

2. 부딪힘 위험 감지: 지게차와 작업자(Dynamic Objects)의 이동 방향 벡터와 속도를 계산합니다. 두 객체의 mjData.geom_xpos 간격이 설정된 임계치 이하일 경우 즉각적인 비상 정지(E-Stop) 로직을 실행합니다.

### 3.3 비전 기반 AI 분석 시뮬레이션 (Scenario 3 & 4 기반)

1. PPE(보호구) 인식: 시뮬레이션 내 작업자 모델의 특정 태그(예: 'helmet', 'vest')를 인식하여 미착용 상태를 판별합니다. 미착용 시 Python의 음성 합성 라이브러리를 통해 경고 음성을 송출합니다.

2. 화재 감지: 특정 구역의 픽셀 변화나 가상 센서 값을 통해 초기 불꽃 및 연기를 감지하고, 관제 센터로 비상 신호를 전송합니다.

### 3.4 데이터 로깅 및 분석 (Scenario 5 기반)

1. 실시간 로깅 (Logging): 위험 상황 발생 시 시간, 좌표, 위험 유형, 스냅샷을 CSV 또는 JSON 형태로 자동 저장합니다.

2. 리스크 맵 최적화: 누적된 로그 데이터를 활용하여 위험 발생 빈도가 높은 구역을 히트맵(Heatmap)으로 시각화하고, 순찰 경로를 재최적화하는 피드백 루프를 구현합니다.

## 4. 상세 시뮬레이션 단계 (Execution Steps)

1. 환경 로드: MuJoCo 3.6.0에서 mj_loadXML을 통해 공장 환경과 로봇 모델을 로드합니다.

2. 센서 데이터 추출: Python 환경에서 mjData.sensordata를 통해 실시간 LiDAR 및 이미지 데이터를 획득합니다.

3. AI 로직 적용: 수집된 데이터를 바탕으로 넘어짐, 부딪힘, PPE 미착용 여부를 분류합니다.

4. Action 실행: 위험 판단 시 로봇의 속도를 0으로 제어하거나 경고 알람을 발생시키는 mjData.ctrl 값을 조정합니다.

5. 사후 분석: 시뮬레이션 종료 후 로깅된 데이터를 분석하여 안전 관리 정책 수립을 지원합니다.

## 5. 성공 지표 (KPI)

1. 감지 정확도: 시뮬레이션 내 발생시킨 위험 상황의 95% 이상 식별.
2. 응답 속도: 위험 감지 후 500ms 이내에 경고 조치(Action) 수행.
3. 시스템 안정성: 24시간 연속 가동 시뮬레이션 시 물리 엔진 충돌 오차 1% 미만 유지.
